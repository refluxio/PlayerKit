# Security Policy

## Reporting a Vulnerability

**Do not open a public issue.** Instead:

1. Use [GitHub Security Advisory](https://github.com/refluxio/PlayerKit/security/advisories/new) (preferred).
2. Or email security@reflux.io.

We will respond within 72 hours.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
